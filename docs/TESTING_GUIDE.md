## ⚠️ 常见问题排查

### 问题1：页面空白或报错
**检查：**
- [ ] 后端服务是否运行（端口3001）
- [ ] 前端服务是否运行（端口5173）
- [ ] 浏览器控制台是否有错误

### 问题2：无法登录
**检查：**
- [ ] 用户名密码是否正确
- [ ] 是否选择了正确的角色
- [ ] 员工数据是否已导入

### 问题3：数据不显示
**检查：**
- [ ] 数据库是否初始化（内存模式每次重启重置）
- [ ] API请求是否成功（查看Network面板）

### 问题4：导入失败
**检查：**
- [ ] CSV格式是否正确
- [ ] 编码是否为UTF-8
- [ ] 必填字段是否完整

## 🐛 发现问题怎么办？

1. **记录问题**：
   - 操作步骤
   - 预期结果
   - 实际结果
   - 错误截图

2. **检查日志**：
   - 浏览器控制台（F12）
   - 后端终端输出

3. **尝试修复**：
   - 刷新页面
   - 重新登录
   - 重启服务

4. **寻求帮助**：
   - 提供详细的错误信息
   - 提供复现步骤

# Phase 2 演示数据说明

## 概述

Phase 2 演示数据覆盖两大模块：**360度互评** 和 **绩效面谈记录**，用于功能展示和测试。

## 运行方式

```bash
cd backend

# 插入演示数据（会先清理旧数据再插入）
npm run seed:phase2

# 仅清理数据
npm run seed:phase2:clean
```

> ⚠️ 需要 PostgreSQL 连接（`DATABASE_URL` 环境变量），不支持 Memory DB。

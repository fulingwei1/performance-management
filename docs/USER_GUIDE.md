## 系统概述

ATE 绩效管理系统支持四种角色，每种角色有不同的功能权限：

| 角色 | 核心功能 |
|------|---------|
| 员工 (Employee) | 查看目标、填写进度、查看评分、提交申诉、晋升申请 |
| 经理 (Manager) | 月度打分、目标审批、KPI 分配、团队管理、季度总结 |
| HR | 互评周期管理、晋升审批、数据导入导出、考核发布 |
| GM | 战略目标设定、全局数据查看、组织架构管理 |

## 登录系统

1. 打开系统地址（如 http://localhost:5173）
2. 输入用户名和密码
3. 点击「登录」
4. 系统自动跳转到对应角色的首页

> 💡 **提示**：首次登录建议修改默认密码。

## 员工指南

### 查看我的目标

1. 登录后点击左侧菜单「目标管理」
2. 查看当前年度的个人目标列表
3. 每个目标显示：标题、权重、目标值、当前进度、状态

### 填写目标进度

1. 在目标列表中点击目标
2. 点击「更新进度」
3. 填写当前进度值和说明
4. 点击「提交」

### AI 生成个人目标

1. 点击「新建目标」
2. 选择关联的战略目标
3. 点击「AI 生成」按钮
4. 系统根据战略目标和岗位自动生成建议
5. 修改调整后点击「提交」

### 查看月度评分

1. 点击「月度绩效」
2. 查看每月的评分、评语
3. 如有异议可提交申诉

### 提交绩效申诉

1. 点击「绩效申诉」→「新建申诉」
2. 选择要申诉的绩效记录
3. 填写申诉理由和证据
4. 提交等待处理

### 晋升申请

1. 点击「晋升管理」→「发起申请」
2. 选择目标职级
3. 填写四项总结（可使用 AI 生成）：
   - 业绩总结
   - 技能总结
   - 能力总结
   - 工作总结
4. 点击「提交」

### 同事互评

1. 点击「同事互评」
2. 在互评周期内选择要评价的同事
3. 填写评分和评语（可使用 AI 辅助）
4. 选择匿名或实名
5. 提交评价

### 查看通知

1. 点击右上角通知图标（🔔）
2. 查看未读通知
3. 点击通知跳转到相关页面

## 经理指南

### 月度绩效打分

1. 点击「月度绩效」→「团队打分」
2. 选择月份和员工
3. 填写评分（0-100）
4. 三种评价方式任选：
   - **手动输入**：直接编写评语
   - **AI 生成**：点击「AI 生成评价」，系统根据分数自动生成
   - **关键词选择**：从 144 个关键词（7 大类）中选择，自动组合成评语
5. 点击「提交」

> 💡 **关键词分类**：工作能力、团队协作、创新思维、责任心、学习成长、沟通表达、问题处理，每类含正向和负向关键词。

### 目标审批

1. 点击「目标审批」
2. 查看下属提交的目标
3. 审核目标合理性
4. 点击「通过」或「驳回」（附评论）

### KPI 分配

1. 点击「KPI 管理」→「分配 KPI」
2. 选择员工
3. 设置 KPI 名称、目标值、权重
4. 保存

### 季度总结

1. 点击「季度总结」→「填写总结」
2. 选择季度
3. 填写总结内容（可 AI 辅助）
4. 提交

### 团队仪表板

1. 点击「仪表板」
2. 查看团队整体绩效数据：
   - 平均分
   - 分数分布
   - 目标完成率
   - 趋势图

## HR 指南

### 互评周期管理

1. 点击「同事互评」→「周期管理」
2. 创建新的互评周期
3. 设置起止日期
4. 发布周期通知

### 晋升审批

1. 点击「晋升管理」→「待审批」
2. 查看申请详情
3. 审核四项总结
4. 审批通过或驳回

### 数据导入

1. 点击「数据管理」→「导入」
2. 下载 Excel 模板
3. 按模板填写数据
4. 上传 Excel 文件
5. 预览导入数据
6. 确认导入

**支持的导入类型：**
- 员工信息批量导入
- 绩效数据批量导入

### 数据导出

1. 点击「数据管理」→「导出」
2. 选择导出类型和筛选条件
3. 选择格式（Excel / CSV）
4. 点击「导出」下载文件

### 考核发布

1. 点击「考核管理」→「发布结果」
2. 选择考核周期
3. 设置发布范围
4. 点击「发布」（自动发送通知）

### 审计日志查看

1. 点击「系统管理」→「审计日志」
2. 按操作类型、用户、日期筛选
3. 查看操作详情

## GM 指南

### 设定战略目标

1. 点击「战略目标」→「新建」
2. 填写目标标题、描述
3. 设置年度、目标值
4. 发布目标

### 组织架构查看

1. 点击「组织架构」
2. 查看可视化组织树（ReactFlow 渲染）
3. 支持缩放、拖拽
4. 点击节点查看部门/人员详情

### 绩效分析

1. 点击「绩效分析」
2. 查看多维度分析：
   - **总览**：全公司绩效概况
   - **趋势**：月度/季度趋势图
   - **部门对比**：各部门绩效排名
   - **分布**：绩效等级分布

### 全局仪表板

1. 点击「仪表板」
2. 查看全公司关键指标：
   - 员工总数、部门数
   - 平均绩效分
   - 目标完成率
   - 待处理审批数

## 常见问题 FAQ

### Q1: 忘记密码怎么办？

联系 HR 或系统管理员重置密码。

### Q2: AI 生成功能无法使用？

检查：
1. 系统是否配置了 AI API Key
2. 网络是否可访问 AI 服务
3. 联系管理员查看 AI 使用配额

### Q3: 月度评分提交后能修改吗？

经理可在当月内修改已提交的评分。跨月修改需 HR 权限。

### Q4: 互评是否真的匿名？

选择匿名后，被评价人无法看到评价者姓名。但 HR 和 Admin 可查看完整记录。

### Q5: 导入 Excel 报错？

常见原因：
- 文件格式不是 .xlsx
- 列名与模板不匹配
- 必填字段为空
- 员工编号重复

建议下载最新模板重新填写。

### Q6: 如何查看历史绩效？

在「月度绩效」页面切换年份和月份即可查看历史记录。

### Q7: 晋升申请被驳回后能重新提交吗？

可以。修改内容后重新提交即可。

### Q8: 系统支持手机访问吗？

系统前端采用响应式设计，支持手机浏览器访问，但推荐使用电脑操作。

### Q9: 数据导出支持哪些格式？

支持 Excel (.xlsx) 和 CSV 格式。

### Q10: 如何联系技术支持？

联系系统管理员或 IT 部门。

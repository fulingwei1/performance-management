## 📋 关键词分级使用建议

### 基层员工（初/中级）
**重点维度**:
- 工作能力（专业、效率、质量）
- 工作态度（主动、责任、投入）
- 团队协作（配合、沟通）
- 学习成长（学习快、进步明显）

**常用正面词**: 积极主动、学习快、配合度高、认真负责、执行力强  
**常用负面词**: 能力欠缺、效率低、被动消极、粗心大意、拖延症

## 💡 使用技巧

### 1. 组合使用原则
- **突出特点**: 选择3-5个最突出的标签
- **正负结合**: 可以正面为主，负面1-2个作为改进方向
- **避免矛盾**: 不要选择相互矛盾的标签（如"积极主动"+"被动消极"）
- **分层次**: 根据员工层级选择合适的关键词

### 2. 评价示例

**示例1: 优秀基层员工**
- ✅ 积极主动、学习快、执行力强、质量优、配合度高
- ⚠️ 经验不足（改进方向）

**示例2: 待提升员工**
- ✅ 服从安排、勤奋努力
- ⚠️ 效率低、粗心大意、沟通不畅

**示例3: 优秀管理者**
- ✅ 领导力强、善带团队、决策果断、目标清晰、前瞻思维
- ⚠️ 授权不够（改进方向）

### 3. 标签数量建议
- **基层员工**: 3-5个（2-3正面 + 1-2负面）
- **骨干员工**: 4-6个（3-4正面 + 1-2负面）
- **管理人员**: 5-7个（4-5正面 + 1-2负面）

## 🎯 系统集成建议

### 前端实现方式

#### 1. 标签选择器
```typescript
// 关键词数据结构
interface Keyword {
  id: string;
  text: string;
  category: string;  // 分类
  type: 'positive' | 'negative';  // 正面/负面
  level: string[];  // 适用层级 ['basic', 'senior', 'manager', 'executive']
}

// 使用场景
<KeywordSelector
  value={selectedKeywords}
  onChange={setSelectedKeywords}
  maxCount={7}
  employeeLevel="basic"  // 根据员工级别筛选关键词
/>
```

- **正面标签**: 绿色背景（bg-green-100 text-green-700）

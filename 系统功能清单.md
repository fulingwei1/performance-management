## 🧪 可测试功能

### 2. 月度绩效管理
**角色**: Manager → Employee
- 经理每月给下属打分
- 填写工作评价
- 使用AI生成评价意见
- 使用关键词快速评价(144个关键词库)
- 员工查看自己的月度得分

**测试路径**: 
- Manager登录 → 月度打分 → 选择员工 → 打分+AI评价+关键词选择

---

### 3. 晋升申请
**角色**: Employee → Manager/HR审批
- 员工发起晋升申请
- 填写4个总结(使用AI):
  - 业绩总结
  - 技能总结  
  - 能力总结
  - 工作总结
- 上级审批流程

**测试路径**:
- Employee登录 → 晋升申请 → AI生成4个总结 → 提交
- Manager登录 → 查看晋升申请 → 审批

---

### 4. 同事互评
**角色**: Employee ↔ Employee
- 选择要评价的同事
- 填写评价意见
- 使用AI生成评价
- 匿名/实名选项

**测试路径**:
- Employee登录 → 同事互评 → 选择同事 → AI生成评价 → 提交

---

### 5. 季度工作总结
**角色**: Manager
- 经理填写季度工作总结
- 使用AI生成总结内容
- 提交给上级审阅

**测试路径**:
- Manager登录 → 季度总结 → AI生成 → 提交

---

### 6. 评价关键词库
**功能**: 快速评价辅助工具
- 144个评价关键词
- 7大类别:工作能力、团队协作、创新思维、责任心、学习成长、沟通表达、问题处理
- 正向/负向维度
- 一键选择,自动组合成评价文本

**测试路径**:
- Manager打分时 → 点击关键词按钮 → 选择关键词 → 自动生成评价

---

### 7. AI使用统计
**角色**: Employee(个人) / Admin(全局)
- 查看个人AI使用情况:
  - 总调用次数
  - 成功/失败次数
  - 总token消耗
  - 总成本(元)
  - 最后使用时间
- Admin查看全公司AI使用排行

**测试路径**:
- Employee登录 → 我的AI使用统计
- Admin登录 → AI使用统计(全员)

---

### 8. 目标确认流程
**角色**: Manager → Employee
- 经理设定目标后,员工确认
- 员工可以填写反馈意见
- 使用AI生成反馈

**测试路径**:
- Manager创建目标 → Employee登录 → 目标确认 → AI生成反馈 → 确认

---

### 9. Dashboard仪表板
**角色**: 所有角色
- Employee Dashboard:
  - 本月得分
  - 待完成任务
  - 目标进度
  - 战略目标总览
- Manager Dashboard:
  - 团队平均分
  - 待打分员工
  - 目标审批
- GM Dashboard:
  - 全公司数据概览
  - 战略执行进度

**测试路径**:
- 各角色登录 → 查看Dashboard数据

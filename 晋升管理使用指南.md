## 功能概述

### 什么是晋升管理系统？

晋升管理系统是 ATE 绩效管理平台的核心模块之一，支持员工晋升与加薪的全流程电子化管理。

### 核心功能

✅ **员工自主申请** - 员工可以提交晋升/加薪申请  
✅ **多级审批流程** - 经理 → 总经理 → HR 三级审批  
✅ **AI 智能助手** - 辅助撰写晋升理由和证明材料  
✅ **历史记录追踪** - 完整记录申请和审批历史  
✅ **数据导出** - 支持导出审批记录（Excel）

### 适用场景

- 📈 **职级晋升** - 从初级到中级、高级的晋升
- 💰 **薪资调整** - 年度调薪、特殊加薪
- 🏆 **岗位晋升** - 晋升到更高职位
- ⭐ **绩优奖励** - 优秀员工的激励晋升

## 晋升流程

### 完整审批流程

```
┌─────────────┐
│ 员工提交申请 │
└──────┬──────┘
       ↓
┌─────────────┐
│ 直属经理审批 │  ← 第一关
└──────┬──────┘
       ↓
┌─────────────┐
│ 总经理审批   │  ← 第二关
└──────┬──────┘
       ↓
┌─────────────┐
│  HR 审批     │  ← 第三关
└──────┬──────┘
       ↓
┌─────────────┐
│  通过/拒绝   │
└─────────────┘
```

### 审批权限

| 角色 | 可以审批的申请 |
|------|---------------|
| 直属经理 | 本部门下属的申请 |
| 总经理 | 经理已批准的申请 |
| HR | 总经理已批准的申请 |

### 状态说明

| 状态 | 说明 |
|------|------|
| 草稿 | 未提交 |
| 待经理审批 | 已提交，等待直属经理审批 |
| 待总经理审批 | 经理已批准，等待总经理审批 |
| 待HR审批 | 总经理已批准，等待HR审批 |
| 已通过 | HR已批准，晋升生效 |
| 已拒绝 | 被任一审批人拒绝 |

## 员工操作

### 1. 访问晋升申请页面

**路径：** 侧边栏 → 晋升加薪

### 2. 提交晋升申请

#### 步骤 1：填写基本信息

1. 点击「+ 新建晋升申请」
2. 系统自动显示当前职级信息
3. 选择目标职级
   ```
   例如：junior（初级）→ intermediate（中级）
   ```
4. 填写目标职位（可选）
   ```
   例如：测试工程师 → 高级测试工程师
   ```
5. 填写期望涨薪比例
   ```
   建议：5%-20%
   注意：涨薪比例需合理，过高可能被拒绝
   ```

#### 步骤 2：填写绩效总结

**要点：**
- 近期绩效表现（最近 3-6 个月）
- 具体成绩和数据
- 与职级要求的对比

**示例：**
```
最近6个月绩效评分：
- 2026年1月：85分（A级）
- 2025年12月：88分（A级）
- 2025年11月：82分（A级）

平均分：85分，连续3个月A级以上，
达到中级工程师的绩效要求。
```

💡 **使用 AI 助手：**
- 点击「绩效总结」右侧的 🤖 图标
- AI 自动生成专业表述
- 可以多次生成，选择最好的版本

#### 步骤 3：填写技能总结

**要点：**
- 掌握的专业技能
- 通过的认证/培训
- 技能提升情况

**示例：**
```
1. 掌握自动化测试框架 Selenium、Playwright
2. 通过 ISTQB 高级测试工程师认证
3. 熟练使用性能测试工具 JMeter
4. 能够独立设计测试方案
5. 指导2名初级测试工程师
```

💡 **使用 AI 助手：**
- AI 可以帮你规范技能描述
- 突出重点技能

#### 步骤 4：填写能力总结

**要点：**
- 工作能力（独立性、解决问题能力）
- 管理能力（如有）
- 学习能力
- 团队协作能力

**示例：**
```
1. 能够独立完成复杂项目的测试工作
2. 主导过 3 个大型项目的测试方案设计
3. 协助经理培训新员工
4. 积极参与部门技术分享
5. 多次获得客户好评
```

#### 步骤 5：填写工作总结

**要点：**
- 主要工作成果
- 项目贡献
- 特别成就

**示例：**
```
1. 2025年度完成 15 个项目的测试工作
2. 发现并跟踪 200+ 个缺陷，缺陷发现率 95%
3. 主导开发自动化测试框架，提升效率 40%
4. 获得部门"年度优秀员工"称号
5. 零客户投诉记录
```

#### 步骤 6：提交申请

1. 检查所有信息
2. 点击「提交申请」
3. 系统发送通知给直属经理
4. 等待审批

### 3. 查看申请状态

**路径：** 晋升加薪页面 → 我的申请列表

**可以看到：**
- 申请状态（待审批/已通过/已拒绝）
- 当前在哪个审批环节
- 审批人的反馈意见

### 4. 撤回申请

如果申请尚未审批，可以撤回重新修改：
1. 在申请列表中点击「撤回」
2. 重新编辑内容
3. 再次提交

⚠️ **注意：** 已经有审批人审批的申请无法撤回

## 经理操作

### 1. 访问审批页面

**路径：** 侧边栏 → 晋升加薪

### 2. 查看待审批申请

**显示内容：**
- 下属提交的晋升申请列表
- 申请人基本信息
- 当前职级 → 目标职级
- 期望涨薪比例
- 提交时间

### 3. 审批流程

#### 查看申请详情

点击申请，查看完整内容：
- 绩效总结
- 技能总结
- 能力总结
- 工作总结
- 最近绩效记录（自动显示）

#### 评估申请

**评估标准：**

1. **绩效要求**
   - 是否达到目标职级的绩效标准？
   - 最近 6 个月平均分是否达标？
   - 是否有低于 C 级的记录？

2. **技能要求**
   - 是否掌握目标职级所需技能？
   - 是否有相关认证？
   - 是否能独立完成相应工作？

3. **工作贡献**
   - 工作成果是否突出？
   - 是否有特别贡献？
   - 是否能带领团队？

4. **涨薪合理性**
   - 涨薪比例是否合理？
   - 是否符合公司薪酬政策？

#### 做出决定

**✅ 通过申请**
1. 点击「通过」
2. 填写审批意见（可选）
   ```
   示例：
   该员工表现优秀，技能扎实，完全达到中级
   工程师要求。建议通过晋升，涨薪10%合理。
   ```
3. 提交，申请流转到总经理

**❌ 拒绝申请**
1. 点击「拒绝」
2. 必须填写拒绝理由
   ```
   示例：
   该员工工作表现良好，但技能水平尚未达到
   中级要求。建议再工作6个月，提升自动化
   测试能力后再申请。
   ```
3. 提交，申请结束

💡 **使用 AI 助手：**
- AI 可以帮你生成专业的审批意见
- 点击审批意见框右侧的 🤖 图标

### 4. 查看审批历史

**路径：** 晋升加薪页面 → 审批历史

可以查看：
- 已审批的申请
- 审批结果
- 审批时间

## 总经理操作

### 1. 访问审批页面

**路径：** 侧边栏 → 晋升审批

### 2. 审批经理已批准的申请

**审批重点：**
1. 检查经理的审批意见
2. 评估涨薪比例是否合理
3. 考虑公司整体预算
4. 与同级别员工对比

**审批流程与经理相同：**
- 通过 → 流转到 HR
- 拒绝 → 申请结束（需填写理由）

## HR操作

### 1. 访问审批页面

**路径：** 侧边栏 → 晋升审批

### 2. 最终审批

**审批重点：**
1. 检查薪酬政策合规性
2. 确认职级调整符合标准
3. 核实绩效记录
4. 考虑公司薪酬预算

**✅ 通过申请（晋升生效）**
1. 点击「通过」
2. 填写审批意见
3. 提交后：
   - 系统自动更新员工职级
   - 更新薪资信息（如需手动调整薪资，在员工管理中操作）
   - 发送通知给员工

**❌ 拒绝申请**
1. 点击「拒绝」
2. 填写拒绝理由
3. 提交，发送通知给员工

### 3. 晋升记录管理

**路径：** 晋升审批页面 → 审批历史

**功能：**
- 查看所有晋升记录
- 筛选（按时间、部门、职级）
- 导出 Excel

**导出字段：**
- 员工姓名
- 部门
- 原职级 → 目标职级
- 涨薪比例
- 申请时间
- 审批时间
- 审批结果

## 常见问题

### Q1: 我可以提交多个晋升申请吗？
**A:** 可以，但建议一次只提交一个。如果有多个未处理的申请，审批人可能会困惑。

### Q2: 晋升申请被拒绝后多久可以再申请？
**A:** 建议至少间隔 3-6 个月，并根据拒绝理由改进后再申请。

### Q3: 涨薪比例写多少合适？
**A:** 建议：
- 正常晋升：5%-15%
- 跨级晋升：15%-30%
- 根据公司政策和行业标准调整

### Q4: AI 生成的内容可以直接用吗？
**A:** 建议作为参考，结合实际情况修改。AI 不知道你的具体工作，需要你补充细节。

### Q5: 审批需要多长时间？
**A:** 
- 经理审批：1-3 个工作日
- 总经理审批：3-5 个工作日
- HR 审批：2-3 个工作日
- 总计：1-2 周

### Q6: 晋升通过后什么时候生效？
**A:** HR 审批通过后立即生效。薪资调整一般从下个月开始。

### Q7: 可以申请跨级晋升吗？
**A:** 理论上可以，但难度较大，需要有非常突出的表现和充分的理由。

### Q8: 经理审批通过后，我还需要做什么？
**A:** 不需要，系统会自动流转到总经理审批，你只需等待即可。

### Q9: 如何撤回已提交的申请？
**A:** 在「我的申请」列表中点击「撤回」（仅限未审批的申请）。

### Q10: 晋升申请失败会影响考核吗？
**A:** 不会。晋升申请是独立的，不影响正常绩效考核。
